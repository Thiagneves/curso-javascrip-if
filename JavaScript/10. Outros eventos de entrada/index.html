Eventos de input

Log de ocorrências

Input


onBlur: Quando sai do campo (tipo sair do campo com o mouse)
onChange: Quando muda o conteúdo do campo
onFocus: Quando entra no campo
onSelect: Quando um texto é selecionado
onSubmit: quando o formulário é enviado
onReset: Quando o botão reset é clicado
onKeyDown: Quando pressionada a tecla (fica com a tecla pressionada. ex: setas)
onKeyUp: Quando solta a tecla
